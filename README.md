# Registro de Glicose - Premium Android App

Uma aplicação Android moderna, minimalista e funcional para o controle e monitoramento de níveis de glicose no sangue. Desenvolvida com as tecnologias mais recentes do ecossistema Android para oferecer uma experiência fluida e profissional.

## 🚀 Funcionalidades

### 📊 Dashboard Inteligente
- **Navegação Contextual**: Navegue entre datas usando setas ou um seletor de calendário.
- **Gráfico de Tendência**: Visualização dinâmica do comportamento da glicose ao longo do dia com gradientes modernos.
- **Resumo do Dia**: Veja rapidamente sua última medição e a lista completa do dia selecionado.

### 📈 Relatórios Clínicos (Insights)
- **Cálculos Automáticos**: Média glicêmica e estimativa de Hemoglobina Glicada (eA1c).
- **Tempo no Alvo (Time in Range)**: Gráfico de distribuição percentual entre níveis Alvo, Altos e Baixos.
- **Análise de Período**: Filtros rápidos para 7 dias, 30 dias ou histórico completo.

### ⏰ Lembretes estilo Alarme
- **Notificações Críticas**: Alertas de alta prioridade que funcionam mesmo com o app fechado.
- **Interface de Alarme**: Pop-ups em tela cheia (Full Screen Intent) para garantir que você não perca o horário da medição.
- **Deep Linking**: Toque na notificação para abrir o app diretamente no campo de registro.

### 📥 Exportação de Dados
- **Exportação CSV**: Gere planilhas profissionais dos seus dados.
- **Compartilhamento Direto**: Salve o arquivo na pasta Downloads e abra o menu de compartilhamento em um único clique.

## 🛠 Tecnologias Utilizadas

- **Kotlin**: Linguagem principal.
- **Jetpack Compose**: UI moderna e declarativa.
- **Material 3**: Design system de última geração com suporte a temas dinâmicos.
- **Room Database**: Persistência de dados local segura e eficiente.
- **Android Architecture Components**: ViewModel, StateFlow e Navigation Compose.
- **AlarmManager & Notifications**: Sistema de alertas críticos e precisos.

## 🛠 Como Compilar

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/RegistroGlicose.git
   ```
2. Abra o projeto no **Android Studio Hedgehog** (ou superior).
3. Certifique-se de ter o **JDK 17** configurado.
4. Sincronize o Gradle e execute no seu dispositivo ou emulador.
